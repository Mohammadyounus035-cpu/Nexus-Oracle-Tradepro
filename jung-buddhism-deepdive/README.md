# Carl Jung and the Buddhist Mirror: A Deep-Dive

This repository contains an investigative report and data-driven visualization scripts mapping the intellectual encounter between Carl Jung and Buddhism from the 1930s to the 1950s. 

It maps Jung's personal realizations—particularly the 1938 dialogue with D.T. Suzuki at Eranos—and analyzes the historical filters (Theosophy, German Romanticism, Colonial translations) through which Buddhist doctrine passed before entering the Western psychological lexicon.

## Repository Contents
* **`generate_visuals.py`**: A Matplotlib script that generates two high-resolution 200 DPI data visualizations and outputs the full Markdown investigative report.
* **`output/`**: The target directory for the generated artifacts.

## Generated Artifacts
When executed, the script produces:
1. `jung_buddhism_deepdive.png` (24x36"): A 7-panel visualization of the historical filters, philosophical conflations, and the mindfulness industrial complex.
2. `96path_jung_buddhism_integration.png` (32x40"): A Sovereign Lattice / 96-Path Map applied to religious and philosophical encounters.
3. `jung_buddhism_report.md`: The complete text report detailing the philosophical score-card and historical analysis.

## Usage
Install the required dependencies:
```bash
pip install matplotlib numpy
```

Run the generator:
```bash
python generate_visuals.py
```
